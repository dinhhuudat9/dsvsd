# Deploy trên Render

Đây là site tĩnh, chỉ cần:

1. Đưa toàn bộ thư mục này lên GitHub/GitLab.
2. Trên Render chọn **New > Static Site** hoặc dùng Blueprint `render.yaml`.
3. Nếu dùng Blueprint, trỏ tới file `render.yaml` trong repo.
4. Render sẽ dùng `index.html` ở thư mục gốc để xuất bản.

Nếu muốn đổi tên site hoặc cấu hình domain, sửa trong Render Dashboard.
